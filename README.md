3 3【APP特点】 编辑本段
　　    1. CC为您呈现当天的世界杯赛事，并叫醒您看球；
　　    2. CC锁屏放置五个您使用频率最高的应用图标；
　　    3. 每次解锁屏幕，CC会为您清理内存，让手机“飞起来”；
　　    4. 锁屏闪电拍照，帮助您在锁屏快速照（tou）相（pai）；
　　    5. 累计开屏，记录您与CC一起相处的时间；
　　    6. 按场景自动切换锁屏背景，让您每秒都有好心情；
　　    7. 支持全屏幕任意位置滑动解锁，方便您单手操作；
